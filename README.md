# UN Portfolio Website

## Overview
This is a Next.js (Pages Router) portfolio website designed for a high-ranking UN official.

## Tech Stack
- **Framework**: Next.js (Pages Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React

## Setup

1. **Install Dependencies** (If not already running):
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   npm start
   ```

## Project Structure
- `/pages`: Application routes (`index.tsx`, `_app.tsx`, `_document.tsx`)
- `/components`: Reusable UI components
- `/styles`: Global styles and Tailwind configuration

## Fonts
- **Headings**: Lexend
- **Body**: Funnel Sans
- **Metadata**: Courier Prime

## Customization
- Update `components/Hero.tsx` with the real portrait image.
- content in `pages/index.tsx` for Roles and Initiatives.
